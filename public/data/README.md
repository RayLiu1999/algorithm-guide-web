## 學習建議

1. **按順序學習**：從 Array → Design，前面的知識是後面的基礎
2. **先看套路模板**：每個分類開頭都有通用模板，先記住框架
3. **理解思路再看程式碼**：不要直接看 code，先嘗試用套路解
4. **反覆練習**：每個分類做完後，隔幾天再回來複習
